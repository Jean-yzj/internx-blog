# 暑期實習 vs 學期實習：時間管理與選擇策略

「我該選擇兩個月的暑期實習，還是一整個學期的兼職實習？」這個問題沒有標準答案，因為兩者各自代表不同的成長路徑。本篇將從時間結構、學習深度、企業偏好等角度，協助你做出最適合自己的選擇。

## 一、暑期實習的特性

暑期實習通常為 **6 月中至 8 月底**，每週 4–5 天全職投入。

### 優點
- **高密度的學習**：8–10 週全職投入，等於濃縮一年的工作量。
- **完整專案**：通常會給你一個獨立 ownership 的小專案。
- **可全心投入**：沒有課業干擾，能完全沉浸在工作環境。
- **大廠多以暑期為主要招募**：頂尖企業的旗艦計畫幾乎都是暑期形式。
- **Return Offer 直接**：暑期實習結束時即決定是否錄取為正職。

### 缺點
- **時間壓縮**：從 onboarding 到產出時間有限。第一週還沒上手，第二週就要開始產出。
- **小專案侷限**：許多公司暑期實習生只跟一個小專案，難以看到產品的全貌。
- **節奏緊張**：壓力可能很大，特別是在頂尖企業。
- **無法觀察長期**：你看到的是 8 週的快照，不是公司的真實樣貌。

### 適合的人
- 希望體驗「正職感」、想搶 Return Offer。
- 對單一公司有強烈興趣的學生。
- 學期內課業壓力大，無法兼顧實習。
- 經濟條件能負擔暑期不打工的學生。

### 暑期實習的「黃金 8 週」節奏

| 週次 | 主要任務 | 心態建議 |
|------|----------|----------|
| 第 1 週 | 認識團隊、學習工具、設定目標 | 多問少做 |
| 第 2 週 | 接手第一個小任務 | 把基本功做穩 |
| 第 3–4 週 | 深入專案核心 | 開始展現主動性 |
| 第 5 週 | 中期檢視 + 調整方向 | 主動約 1on1 |
| 第 6–7 週 | 完成主要交付 | 全力衝刺 |
| 第 8 週 | 簡報成果 + 拿 Return Offer 訊號 | 收尾 + 道別 |

## 二、學期實習的特性

學期實習通常為 **每週 2–3 天、為期 4–6 個月**。

### 優點
- **時間長**：足以參與長線專案、看到完整產品週期。
- **與團隊建立深度關係**：你會被當成「半個同事」，而不是「臨時實習生」。
- **能反覆觀察公司文化**：4–6 個月看到公司的真實面貌。
- **能驗證 fit**：你能更確定自己是否真的想長期待。
- **轉正機會自然**：許多公司直接把學期實習生視為「準員工」。

### 缺點
- **與課業並行壓力大**：需要強大的時間管理。
- **學期內的事情多**：考試、報告、社團都會搶你的時間。
- **容易心力分散**：兼顧多事的代價是「樣樣都做不到頂」。
- **薪資較低**：因為非全職，月薪相對暑期實習低。

### 適合的人
- 想累積「真正會做的能力」。
- 希望探索特定產業深度。
- 修課負擔較輕的學生。
- 已有工作節奏與時間管理能力的學生。
- 想在實習中觀察「文化是否合拍」的人。

## 三、企業如何看待這兩者？

不同產業偏好不同形式：

### 顧問業、投資銀行
- 以暑期實習為主，且往往是 Return Offer 來源。
- 旗艦級暑期實習（Summer Analyst、Summer Associate）是進入這個產業的「主要門票」。
- 例：BCG、麥肯錫、Bain、Goldman Sachs、Morgan Stanley。

### 科技業 PM／工程
- 暑期實習為主，但學期實習也越來越多開放。
- FAANG 的 Internship Program 多為暑期 12 週。
- 台灣科技公司（如 Dcard、iKala、KKLab）多開放學期實習。

### 行銷、廣告、創意產業
- 學期實習很常見，重視持續性而非短時間爆發。
- 媒體公司、廣告代理商、品牌行銷部門較喜歡長期累積。
- 例：奧美、李奧貝納、4A 廣告公司。

### 新創公司
- 兩者都歡迎，但更喜歡能長期承擔任務的人。
- 學期實習轉正比例極高（70–80%）。

### 製造業 / 半導體
- 多數採用「實習生計畫」，可能跨學期 + 暑期。
- 例：台積電、聯發科。

## 四、時間管理：學期實習的關鍵

學期實習若沒有規劃好，會讓 GPA 與心理健康同時崩盤。

### 1. 預估每週投入時間
實習 + 通勤 + 額外加班，至少 20 小時起跳。

學期內合理的「總投入時間」：
- 實習：20 小時
- 課業：30 小時（含上課、寫作業、念書）
- 社團 / 個人專案：5–10 小時
- 睡眠：56 小時（每天 8 小時）
- 其他（吃飯、運動、社交）：剩餘時間

**Reality check**：每週 168 小時，扣除前述後只剩 50–60 小時餘裕。如果還想保持運動與社交，學期實習要做得有效率。

### 2. 重課與實習分開週
避開期中期末考週，先和主管討論彈性安排。
- 提前 2 週告知主管考試時間。
- 大型 deadline 不要同時跨在考試週。

### 3. 每週固定回顧
用 30 分鐘檢視這週學到什麼、下週哪些可優化。
- 把學到的東西寫進個人筆記。
- 把優化清單帶進下週 1on1。

### 4. 建立「能量管理」而非「時間管理」
時間人人 24 小時，但能量會隨壓力衰退。建議：
- 早上做最重要的事（高能量時段）。
- 晚上處理低耗能任務（回 email、整理資料）。
- 週日完全休息一天，避免 burnout。

## 五、暑期實習的隱性挑戰

很多學生以為「暑期就是把全部時間給公司」，但更聰明的做法是：

### 1. 第一週快速融入
- 主動約 1on1（與主管、團隊夥伴）。
- 自備筆記，記下每個術語與流程。
- 不怕問問題（第一週「無罪期」是黃金時段）。

### 2. 設定 60 天目標
和主管討論「實習結束時你希望我達到什麼？」。把答案寫下，每週對照進度。

### 3. Return Offer 邏輯
不是「乖乖把事情做完」，而是「主動超出期待」。
- 主動提出問題的解方，而非只報告問題。
- 主動承擔小型額外專案。
- 主動與其他部門合作建立網絡。

### 4. 第 5 週的中期檢視
這是黃金分水嶺。主動約主管 30 分鐘聊：
- 「我目前的表現符合您的期待嗎？」
- 「有哪些地方我還可以加強？」
- 「我想把目前的專案延伸到 X 方向，您覺得可行嗎？」

### 5. 第 8 週的收尾簡報
製作一份完整的「實習成果簡報」（10–15 張投影片）：
- 我做了什麼。
- 帶來什麼影響。
- 學到什麼。
- 接下來的建議。

這份簡報會留在公司，成為決定 Return Offer 的關鍵之一。

## 六、混合策略：一年兩段累積

對追求極致履歷的學生來說，可以考慮：
- **大三上 + 寒假**：學期實習 + 短期專題，累積基本功。
- **大三暑假**：旗艦級暑期實習，鎖定目標公司。
- **大四上學期**：延續暑期實習進入學期 part-time，提升 Return Offer 機率。

### 實際排程範例

| 時間 | 學業 | 實習 |
|------|------|------|
| 大二上 | 修核心課 | 無 |
| 大二下 | 修選修 | 學期 part-time（10 hrs/週） |
| 大二暑 | 暑期 | 短期實習（4–6 週） |
| 大三上 | 修專業課 | 學期 part-time（15 hrs/週） |
| 大三下 | 修專業課 | 無（準備暑期申請） |
| 大三暑 | 暑期 | 旗艦級暑期實習（8–10 週） |
| 大四上 | 修選修 | 學期 part-time（轉正準備） |
| 大四下 | 寫論文 | 無或長期 part-time |

## 七、做選擇前的三個提問

### 1. 這個學期的課業負荷是否能負擔學期實習？
看你這學期修的學分與必修課比例。20 學分 + 重修 + 期末論文 ≠ 適合。

### 2. 這家公司的暑期實習有 Return Offer 制度嗎？
若有，暑期實習是更直接的選擇。若沒有，學期實習可能更值得（因為你能慢慢累積關係）。

### 3. 你想要的是「廣度體驗」還是「深度累積」？
- 廣度：每段都不長、嘗試不同產業 → 適合暑期實習。
- 深度：固定一個方向、長期累積 → 適合學期實習。

## 八、產業別的「典型實習路徑」

### 顧問業學生
- 大三暑假：頂尖顧問 Summer Internship（必爭）。
- 大四上：可能不再實習，準備正職面試。

### 投資銀行學生
- 大三暑假：Summer Analyst（決定一切）。
- 大四上：Off-cycle Internship（補實習）。

### PM 學生
- 大二暑：小新創 PM 助理。
- 大三暑：大型科技公司 PM Internship。
- 大四上：學期 part-time（轉正準備）。

### 行銷學生
- 大二開始累積：每學期一段 part-time。
- 大三暑：品牌方暑期實習。
- 大四上：學期 part-time（深化專業）。

### 工程師學生
- 大一起：Side Project + Hackathon。
- 大二暑：小公司或新創實習。
- 大三暑：大廠 Internship（FAANG、Dcard、台積等）。
- 大四：通常已有 Return Offer，可選擇是否再實習。

## 九、不適合學期實習的訊號

- 你這學期修了 22 學分以上。
- 你的 GPA 已經低於門檻，需要提升。
- 你有重要的競賽或考試（如 GRE、JLPT、CFA Level 1）。
- 你的健康狀況不穩定。
- 你正在經歷重大個人事件。

**重要原則**：實習是工具，不是目的。為了實習犧牲健康或畢業是不值得的。

## 十、FAQ

**Q1：我可以同時做暑期 + 學期實習嗎？**
可以，但要確保是同一家公司延續。同時兩家會違反多數實習合約的「專屬條款」。

**Q2：學期實習的薪資合理嗎？**
台灣學期實習薪資通常為 180–250 元/小時。低於 180 元就要留意是否合理。

**Q3：如果我已經有暑期實習了，還要再做學期嗎？**
看你的目的：補充新技能 → 做；只是怕履歷空 → 不必。

**Q4：實習可以中途辭職嗎？**
不建議。除非有重大個人因素，否則完整做完是基本職場倫理，也會影響推薦人關係。

**Q5：暑期實習比較重要還是學期實習？**
**頂尖企業看暑期，深度學習看學期**。兩者價值不同。

## 結語

暑期或學期實習都不是萬靈丹，**真正的差距是你怎麼運用每一段時間**。同樣兩個月，有人完成兩份專案、有人只是當打卡型助理。

**核心原則**：
1. **匹配你的目標**：你的目標公司偏好哪一種？
2. **配合你的學期**：不要為了實習毀了 GPA。
3. **設定「離開時要帶走什麼」**：每段實習都要有可量化的成果。
4. **動態調整**：每年根據新狀況重新規劃，不要綁死路徑。

**行動建議**：到實習通搜尋你目標公司，閱讀「暑期實習」與「學期實習」的不同心得，比較哪一種模式更符合你想要的成長節奏。再評估自己這學期的課業負荷，決定下一段最適合的形式。
