# 電商 / D2C 品牌：行銷與營運實習機會地圖

「電商實習是不是只能做客服？」「進蝦皮、momo 是不是比較有保障？」「D2C 品牌沒有名氣，去那邊會不會浪費時間？」這是學生社群中關於電商最常被問到的三個問題。台灣電商產業近年進入「平台戰已定、品牌力決勝負」的階段：大型平台拼物流與會員、中型品牌拼受眾經營、新創 D2C 拼產品故事。本篇文章將帶你看清電商生態地圖、實習職類分布、必備工具與長期成長路徑，幫你決定要進哪一段。

> **閱讀前請先知道**：台灣電商平台變動非常頻繁（Yahoo 超級商城已收、Yahoo 購物中心於 2025 年由 PChome 收購整併、Coupang 酷澎於 2024 年起進攻台灣、TikTok Shop 內容電商崛起）。本文提到的公司名稱、職缺與工具請以**該公司官網最新公告為準**；若文中某平台你在實習通找不到對應職缺，先到該公司全球 / 台灣官網確認是否仍在台灣營運。

## 一、台灣電商生態：平台、品牌、跨境三大區塊

### 1. 大型綜合平台

代表：**蝦皮購物、momo 購物網、PChome 24h、Yahoo 奇摩購物中心（2025 年由 PChome 集團整併）、博客來、Coupang 酷澎（韓系，2024 年起進入台灣）**。

- 商業模式：賣家入駐 + 自營商品 + 廣告變現。
- 規模：年 GMV 數百億新台幣級別；蝦皮、momo 已形成「雙寡頭」格局，PChome 市占率在近年下滑、Yahoo 購物中心被 PChome 整併、Coupang 主打快速到貨。
- 文化：大公司制度、跨部門複雜、流程多。
- **變動提醒**：以上市占與整併狀況變動頻繁，投履歷前先看該公司最新季報 / 新聞。

### 2. 中型品牌型電商

代表：誠品線上、寶雅線上、康是美網購、各 SPA 服飾品牌官網（如 Lativ、東京著衣 / OB 嚴選等近年仍持續經營者），以及成熟的中型自有品牌官網。

- 商業模式：自有品牌官網 + 平台分銷。
- 規模：中型企業，幾十到幾百人。
- 文化：商業節奏快、行銷與營運是核心。

### 3. 新創 D2C 品牌（Direct-to-Consumer）

代表（皆需以官網最新狀態為準）：
- **食品 / 飲品**：老媽拌麵（陳家私房）、Daylily（漢方保健）、樂維根（蛋白飲）
- **美妝 / 保養**：綠藤生機、TAKAMI、Dr.Wu（旗下 D2C 線）
- **服飾 / 鞋款**：林果良品 ORINGO（手工男鞋）、Pinkoi 上的本土設計師品牌
- **寵物 / 家居 / 健康食品**：多為近年募資 / 群募起家的新銳品牌

- 商業模式：自建官網（多用 SHOPLINE / Cyberbiz / 91APP / Shopify）+ 社群行銷 + 廣告投放。
- 規模：通常 10–50 人。
- 文化：扁平、節奏快、實習生上手快。

### 4. 跨境電商

代表：在 Amazon、Shopee 海外站（新馬、印尼、泰國、越南、菲律賓、巴西）、樂天日本、Coupang 韓國等市場銷售的台灣賣家。

- 商業模式：跨境物流 + 本地市場行銷。
- 規模：差異大，從個人賣家到中型公司。
- 文化：英文 / 多語系是基本，海外節奏快。

## 二、實習職類拆解

### 1. 行銷（Marketing）

- 廣告投放：Meta Ads（FB / IG）、Google Ads、TikTok Ads、LINE LAP（LINE Ads Platform，台灣覆蓋約 2,200 萬用戶）。
- 內容行銷：寫文案、製作圖卡、拍短影片。
- 社群經營：IG、Threads、Facebook、TikTok。
- 異業合作：聯名活動、KOL 合作、品牌跨界。

### 2. 營運（Operations）

- 商品上架、SKU 管理、活動檔期規劃。
- 物流與庫存協調、供應商溝通。
- 會員經營與數據分析。

### 3. 客服與用戶體驗（CX）

- 處理客戶問題、退換貨流程、回覆評論。
- 蒐集回饋、提供給產品與行銷部門優化。

### 4. 商業開發（BD / 招商）

- 平台端的「招商 BD」：負責拉品牌入駐。
- 品牌端的「通路 BD」：負責談下實體與線上通路。

### 5. 資料分析（Data Analyst）

- 看廣告 ROAS、看產品銷售結構、看會員行為。
- 對數據敏感、會 Excel + SQL 是基本。

| 大型綜合平台 | 蝦皮、momo、PChome、Coupang | 行銷 / 營運 / BD / 資料 | 中 | 不限科系，商管略多 | 業界中段偏上 |

## 四、必備工具清單

### 1. 廣告投放工具

- **Meta Ads Manager**：FB / IG 廣告必備。
- **Google Ads**：搜尋 + GDN + Performance Max。
- **TikTok Ads Manager**：年輕族群必投；2026 年若 TikTok Shop 在台灣正式營運，會多一個內容電商通路（**依該平台官方公告為準**）。
- **LINE LAP（LINE Ads Platform）**：台灣特殊強勢平台，2026 年起整合 AI 廣告智慧助手。

### 2. 數據工具

- GA4（Google Analytics 4）：流量與轉換分析。
- Looker Studio：資料視覺化。
- Excel + Pivot Table：基本中的基本。
- 簡單 SQL：中型品牌以上會用到。

### 3. 內容與設計工具

- Canva：90% 的 D2C 實習生都用它。
- CapCut：短影片剪輯。
- Notion：專案與內容排程。

### 4. 電商後台（D2C 開店平台 — 容易混淆，請看清差異）

- **SHOPLINE**：總部新加坡 / 香港，擅長社群電商與直播導購，台灣中小品牌常用。
- **Cyberbiz**：台灣品牌，主打 OMO（線上線下整合）與倉儲物流。
- **91APP**：台灣唯一上市的電商平台服務商，定位中大型品牌與連鎖店 App 會員經營。
- **Shopify**：加拿大品牌，App Store 生態最完整、跨境最便利。
- **EasyStore**：總部馬來西亞，多通路整合（東南亞 + 0% 抽成）見長。
- 其他：WACA、Meepshop 等台系平台。

> 這 5 家**很容易被學生互相混淆**——它們服務對象、母國、收費模式差異很大，**面試前請各自到官網看 5 分鐘 pricing 頁**。

賣家中心：蝦皮 / momo / PChome / Coupang 等大型平台各有自己的賣家後台。

EDM / 行銷自動化：Klaviyo、ActiveCampaign、Omnisend、台系 Newsleopard 等。

## 五、平台 vs 品牌：你該怎麼選？

### 大型平台的優點

- 名氣與履歷光環（蝦皮、momo 是知名公司）。
- 制度成熟、學習資源完整。
- 同事多元、跨部門經驗豐富。

### 大型平台的缺點

- 你做的事情可能很「螺絲」，難看到完整全貌。
- 流程冗長，一個專案常需要跑多層審核。
- 實習生較難「拉出個人作品」。

### 中型品牌的優點

- 能看到完整商品週期，從上架到行銷到出貨。
- 有機會接觸品牌策略與創意。

### 中型品牌的缺點

- 沒有大公司的制度與訓練資源。
- 跳槽時 HR 對品牌的認知度低。

### D2C 新創的優點

- 工作範圍最廣，你可能一週做廣告、寫文案、做客服、整理倉庫。
- 跟創辦人距離近，能看到商業決策怎麼做的。
- 成長速度最快。

### D2C 新創的缺點

- 公司可能 1 年後倒閉。
- 沒有現成 SOP，要自己摸索。
- 實習薪資偏低，**部分公司會以股權 / 分紅補貼，依公司條件而定**。

## 六、一日工作範例

### 大型平台行銷實習生

- 9:30 進辦公室，看廣告儀表板。
- 10:00 跟主管 weekly meeting，討論本週檔期。
- 11:00–12:00 整理 KOL 合作的內容素材。
- 13:00–15:00 寫雙 11 活動 landing page 文案。
- 15:00–16:00 跨部門會議，跟營運對齊上架時程。
- 16:00–18:00 看廣告數據、調整出價。

### D2C 新創行銷實習生

- 10:00 看昨晚廣告數據，發現 ROAS 跌了。
- 10:30 跟創辦人 standup，決定下午要換素材。
- 11:00–12:00 用 Canva 做 5 組新廣告圖。
- 13:00–14:00 上 IG 限動 + 回客戶私訊。
- 14:00–16:00 跑去倉庫盤點庫存（同事請假）。
- 16:00–18:00 寫週報、規劃下一檔活動。
- 18:30 下班但回家還會繼續看廣告。

## 七、未來成長路徑

### 路徑 A：平台 → 品牌方

從蝦皮、momo 出來的人很受品牌方歡迎，因為「懂平台規則」。

### 路徑 B：品牌方 → 自創品牌

做過 2–3 個 D2C 品牌的人，最後常自己創業，因為他們已經懂從 0 到 1。

### 路徑 C：行銷 → 顧問 / 廣告代理商

從品牌端跳到代理商，可以服務多個客戶、學更廣。

### 路徑 D：營運 → 跨境 / 海外市場

台灣電商人才在東南亞市場非常搶手，特別是會中文 + 英文的。

### 路徑 E：資料分析 → 產品經理

電商資料分析做久了，可以跳到平台的 PM 角色。

## 八、薪資現實

> 以下為**近年觀察區間**，實際數字請以 104、1111、比薪水及該公司官方招募頁為準。台灣基本工資 2026 年為 NT$29,500 / 月（時薪 NT$196），低於此為違法。

### 起薪

- 大型平台：業界中上，福利齊全。
- 中型品牌：業界中段，紅利依業績。
- D2C 新創：業界中段，可能含股權（**以聘僱合約為準**）。
- 跨境電商：依公司差異大，做美國市場通常較高。

### 5 年後

- 平台行銷經理：業界中上。
- 品牌行銷主管：業界中上 + 業績獎金。
- 自創品牌：天差地遠，從負債到年薪數百萬都有。

## 九、適合什麼樣的你

- **喜歡看數字**：廣告 ROAS、轉換率、客單價是你的日常。
- **能快速產出**：今天的數據明天就要修，不能拖。
- **對消費者行為有好奇心**：為什麼人會在週日晚上下單最多？你會想搞懂。
- **能扛多任務**：D2C 新創一天要切 5 個身分。
- **有美感**：素材好不好看直接決定 ROAS。
- **抗挫折**：90% 的廣告測試都會失敗。

## 十、不適合的訊號

- **無法忍受快速變化**：演算法週週變，你受不了就走不下去。
- **不喜歡接觸消費者**：客服與用戶研究是核心工作。
- **完全不看數據**：純創意走不遠。
- **重度完美主義**：電商行銷是「先發再修」，做完美等於沒做。
- **抗拒「做苦工」**：上架商品、整理庫存、回客訴是不可避免的。
- **無法接受公司可能消失**：D2C 新創的存活率本來就低。

## 十一、結語

電商不是「進蝦皮就贏一半」，而是「你選的賽道決定你能不能持續成長」。大型平台給你光環與制度，中型品牌給你完整視角，D2C 新創給你生死考驗。每一段都是學費，但學到的東西完全不同。

**核心原則**：

1. **先看你想學什麼**：學制度進平台、學品牌進中型、學從 0 到 1 進新創。
2. **數據力 + 美感**雙修：少了一邊都會被淘汰。
3. **接受快節奏**：電商人沒有「等一下」的選項。

**行動建議**：到實習通搜尋「電商」、「行銷」、「D2C」、「蝦皮」、「momo」、「Coupang」、「SHOPLINE」、「Cyberbiz」、「91APP」等關鍵字，比較不同階段公司實習生的真實心得。挑一家大平台 + 一家 D2C 品牌投遞，用兩個暑假體驗兩種極端。等你看過光譜的兩端，就會知道哪一邊是你的長期戰場。
