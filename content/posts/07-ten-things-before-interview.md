# 面試前必做的 10 件事：從研究公司到準備服裝

收到面試通知的瞬間，興奮之餘也是壓力的開始。決定面試成敗的，往往不是當天的表現，而是**面試前 24–72 小時的準備**。本篇整理出十項實戰必備，幫你把臨場焦慮降到最低。

> 註：以下流程整理自台灣本地實習與新鮮人面試經驗（104、Yourator、CakeResume、面試趣 等平台分享）。若是外商或特殊產業（如顧問業 case interview、軟體工程 coding interview），請另外針對該類面試專門準備。

## 一、徹底研究公司（不只是首頁）

不要只看官網的「關於我們」，要往深一層挖：

### 1. 最近 6 個月的新聞

- 產品上線、功能更新。
- 財報、業績預告（若是上市公司，可到**公開資訊觀測站 mops.twse.com.tw**查）。
- 合作消息、併購、組織重整。
- 媒體報導、創辦人訪談（**商業周刊、天下、數位時代、INSIDE、Meet 創業小聚** 是常見來源）。

### 2. 公司年度報告

- 上市公司年報、ESG 報告中能看出策略方向（公開資訊觀測站可下載 PDF）。
- 看「未來計畫」、「風險因子」段落，這些常會出現在面試問題中。

### 3. 創辦人或 CEO 的個人媒體

- LinkedIn 貼文、Medium 文章。
- 訪談 Podcast（如 **股癌、科技島讀、商業思維學院、創新拿鐵**）、TED Talk、Y Combinator 系列影片。
- X / Threads / Facebook 上的觀點分享（台灣中高階主管很多人仍主用 Facebook）。

### 4. 競品分析

- 該公司的 2–3 個主要競爭對手是誰？
- 它們的差異在哪裡？
- 該公司的優勢與劣勢？

### 5. 產品實際使用

- 若是 B2C 產品：註冊一個帳號、實際用 1–2 週。
- 若是 B2B 產品：看 Demo 影片、案例研究（case study）。
- 寫下你的觀察、有共鳴或不順手的地方。

**研究時間建議**：總共 3–4 小時，分散在面試前一週完成。

## 二、研究你的面試官

如果通知信中有揭露面試官姓名，立刻 Google + LinkedIn：

### 必查的資訊

- 學經歷背景（大學、研究所、過往公司）。
- 他在公司負責什麼產品線、團隊大小。
- 他過去發表過的文章、演講、Podcast。
- 興趣（從 LinkedIn 貼文、Medium、個人網站看得出來）。

### 為什麼這很關鍵？

- 如果面試官是同校（台大、政大、清交、成大…）校友 → 可建立連結。
- 如果他過去寫過某主題的文章 → 你能在 Q&A 階段問深度問題。
- 如果他剛加入公司（< 1 年）→ 他可能更開放、更願意分享。

### 進階做法

- 看他在公司的 Title 變動歷程（從 Associate PM → PM → Senior PM → Head of Product），推測他的成長路徑。
- 若他是創辦人，看公司其他公開員工（LinkedIn 搜公司名）有什麼背景，推測團隊文化。

> 提醒：**不要在 LinkedIn 上按到「查看個人檔案」被對方知道**。如果你沒升級 Premium，先在設定中把「個人檔案瀏覽選項」改為匿名模式。

## 三、把 JD 拆成關鍵字清單

把職缺敘述（Job Description）每一句拆成關鍵字。

### 拆解範例

職缺敘述：「需具備數據分析能力、跨部門溝通經驗，熟悉 SQL 與 Excel，能在快節奏環境工作。」

拆解：

- **數據分析能力** → SQL、Excel 樞紐分析、A/B 測試、Tableau、Looker Studio、GA4。
- **跨部門溝通** → 工程協作、設計協作、業務簡報。
- **快節奏環境** → 多工、優先排序、抗壓性。

### 對應你的故事

針對每個關鍵字，想出一段你的故事：

- SQL → 「我在 X 專案中用 SQL 查詢 50 萬筆訂單資料，找出流失客群。」
- 跨部門溝通 → 「我在系學會擔任協調者，整合行銷組與資訊組共同完成活動。」
- 快節奏 → 「我在 4 週內同時參加商業競賽 + 期末考 + 社團活動。」

## 四、準備自我介紹（90 秒版本）

最常見也最容易翻車。

### 三段式結構

**第一段（15 秒）：我是誰**

- 學校 + 系所 + 一句你的特質。
- 例：「我是 OO 大學財金系大三的張小明，是一個喜歡用數據說故事的人。」

**第二段（45 秒）：我做過什麼**

- 一個最相關的經驗 + 量化結果。
- 例：「過去一年我在 OO 公司擔任數據分析實習生，用 SQL 和 Tableau 完成 3 個顧客行為分析專案，其中一個專案幫助公司挽回約 8% 的流失客戶。」

**第三段（30 秒）：我為什麼來這裡**

- 為什麼是這家公司 + 這個職位 + 你能帶來什麼。
- 例：「我特別欣賞貴公司在金融科技的 ESG 投入，我也希望把『用資料挖洞察』的能力帶進這個團隊。」

### 練習方法

- 錄影自己講 90 秒，看回放。
- 注意：是否流暢、有沒有用「呃、嗯、欸」、節奏是否舒服。
- 至少練 5 遍。

## 五、用 STAR 法準備 5–7 個故事

把過去經驗寫成 5–7 個故事卡，每個故事都包含：

- **Situation**：背景情境
- **Task**：你的任務
- **Action**：你的行動
- **Result**：成果與量化數據

### 必備的 7 種故事類型

1. 領導力（你帶過團隊）
2. 衝突解決（你和同事意見不合）
3. 失敗 / 挫折（你失敗過怎麼站起來）
4. 創新 / 改善流程
5. 跨部門合作
6. 在壓力下完成任務
7. 主動承擔超出責任範圍的事情

每個故事控制在 90 秒內可講完。

### 故事卡範例

**主題**：跨部門合作

**Situation**：去年系學會與資訊系合辦 Hackathon，雙方對「主題」與「規格」有強烈分歧。

**Task**：我作為總召，需要在 2 週內讓兩個系協調出共識。

**Action**：

1. 召集雙方代表開「目標對齊會議」，先談大目標再談細節。
2. 建立共享 Notion / HackMD 文件，所有決策可追蹤。
3. 把分歧拆成「可協商」與「不可協商」項目。

**Result**：活動最終吸引 80 人參與，兩系合作後續又延伸出 2 個聯合計畫。

## 六、寫下你要問面試官的問題

至少準備 5 個問題，避免最後說「我沒有問題了」（地雷）。

### 三種類型的問題

**1. 探討文化的問題**

- 「這個團隊最近一年做過最有挑戰的專案是什麼？」
- 「您覺得哪些特質的人在這個團隊最容易成功？」
- 「團隊的 1-on-1（一對一面談）文化如何？多久一次、聊什麼？」

**2. 探討發展的問題**

- 「實習生在這個職位的成長路徑是什麼？」
- 「上一位實習生現在在做什麼？」
- 「公司有提供哪些學習資源（書、線上課、研討會）？」

**3. 探討面試官個人**

- 「您加入公司至今，最有收穫的是什麼？」
- 「您最近一個讓您興奮的工作專案是什麼？」
- 「您看到產業未來 3 年最大的變化會是什麼？」

### 不該問的問題

- 公司基本資訊一查就知道的（薪資、福利可以問，但官網寫得很清楚的就別問）。
- 顯得你想偷懶的問題（「加班頻率？」聽起來像怕辛苦——若真的想問，請改用「請問團隊典型的工作節奏是？」）。

## 七、模擬面試

找 1–2 位學長姐或朋友幫你做完整模擬。也可以利用學校的**職涯中心 / 職涯發展辦公室**（台大、政大、清大、交大、成大、北大、北科、實踐、輔仁等校都有），多數提供免費的模擬面試服務。

### 模擬流程

1. 對方扮演面試官，依 JD 提問 8–10 題。
2. 全程錄影（手機架個三腳架即可）。
3. 模擬完後對方給回饋。
4. 自己看回放，找出 3 個改進點。

### 自我檢視重點

- 肢體語言與眼神接觸。
- 是否說「呃、嗯、那個」過多。
- 講故事時是否清楚明確。
- 情緒是否穩定。
- 結尾是否有完整收尾。

### 進階版：「壓力面試」模擬

請朋友刻意打斷、質疑、追問。這能讓你習慣突發狀況。

## 八、確認服裝與儀容

不同產業差異大（以下為台灣業界常見原則，最終仍以該公司官網照片、員工社群貼文觀察到的風格為準）：

### 1. 金融、顧問、法律、會計師事務所

- 男生：深色西裝（黑、深藍、深灰）+ 素色襯衫（白、淺藍）+ 領帶。
- 女生：套裝（裙裝或褲裝皆可）+ 簡單妝容；裙長至少及膝。

### 2. 科技業（台積電、聯發科、Google、Meta 等）

- 多數可商務休閒（Smart Casual），但**台廠半導體/代工偏正式**，第一次面試仍建議西裝；外商網路公司則放寬。
- 男生：襯衫 + 西裝褲（不必領帶）+ 皮鞋或樂福鞋。
- 女生：襯衫 / 針織衫 + 西裝褲或及膝裙。

### 3. 新創、行銷、媒體、電商

- 商務休閒即可。乾淨整潔比正式更重要。
- 可帶有個人風格但避免太誇張。

### 4. 創意、設計

- 可帶有個人風格但整潔。
- 避免太隨便（如球鞋配短褲）也避免太正式（西裝會顯得格格不入）。

### 5. 傳統製造業、公部門

- 偏正式，建議西裝或正裝。

### 服裝準備時程

- 提前一週確認服裝是否合身。
- 提前一天試穿、整燙。
- 避免買新鞋（容易磨腳）。
- 視訊面試也建議穿正式上身，連褲子也建議穿好（站起來就穿幫）。

## 九、確認交通與設備

### 實體面試

- 提前確認地點，目標**提前 10–15 分鐘抵達**（太早抵達也會讓面試官有壓力）。
- 留心交通（捷運、台鐵、轉乘、台北平日尖峰塞車緩衝至少 30 分鐘）。
- 帶實體履歷紙本 3 份（給面試官 + 自己 + 備用）。
- 帶筆與筆記本。
- 帶身分證（部分公司大樓需要登記訪客）。
- 確認大樓位置：很多台北的軟體公司在內湖科學園區、南港軟體園區，從捷運站走到公司可能還要 10–15 分鐘。

### 視訊面試

- 測試網路、燈光、麥克風、背景（建議乾淨牆面或書櫃）。
- 下載**主流會議軟體**：Google Meet、Microsoft Teams、Zoom。台灣公司目前以 Google Meet 與 Microsoft Teams 為主，Zoom 也常見。
- 充電線備好；筆電也建議全程接電源。
- 通知家人 / 室友不要打擾。
- 預備一個有訊號的 Plan B 地點（家裡網路 down 怎麼辦？學校自習室？星巴克？）。

## 十、面試前一晚的儀式

### 1. 早睡

7 小時以上是常見建議。睡眠不足會讓記憶力與情緒明顯下降。

### 2. 不要熬夜背稿

反而會讓表現僵硬。背過 5 遍就夠了，重點是放鬆。

### 3. 準備物品清單

- 履歷紙本
- 面試官名單 / 公司資料筆記
- 一瓶水
- 手機（記得改靜音）
- 必要文件（成績單、作品集 USB / 雲端連結）

### 4. 隔天早上的儀式

- 提前 1.5 小時起床。
- 5 分鐘深呼吸或散步。
- 一份輕量早餐（避免太油膩、含洋蔥蒜頭的食物）。
- 提前 30 分鐘到達現場附近。
- 在現場附近安靜處（咖啡廳、便利商店座位、樓下大廳）再做 5 分鐘自我介紹練習。

## 十一、進階：面試前 24 小時的「最後檢查清單」

- [ ] 公司近 6 個月新聞已看過。
- [ ] 面試官 LinkedIn 已查過。
- [ ] JD 拆解 + 故事卡準備完成。
- [ ] 自我介紹練習 5 遍以上。
- [ ] 5 個面試問題已寫下。
- [ ] 服裝、配件、儀容已準備。
- [ ] 交通路線確認、備案準備。
- [ ] 設備測試完成（視訊面試）。
- [ ] 履歷紙本印好 3 份（實體面試）。
- [ ] 提前安排好作息，預留 7 小時睡眠。

## 十二、FAQ

**Q1：面試前要不要喝咖啡？**
適量。如果你習慣每天喝，不要當天突然不喝。如果你不習慣，不要為了「提神」突然喝（容易心悸 + 跑廁所）。

**Q2：可以帶筆記進去嗎？**
可以帶簡單的「問題清單」與「故事卡關鍵字」。不要把整本背誦稿帶進去——一旦面試官看到你低頭讀稿，印象會大扣分。

**Q3：如果遲到怎麼辦？**
**立刻**打電話或發訊息給聯絡窗口，誠實道歉並說明預計到達時間。台灣面試文化對準時要求嚴格，遲到本身就會扣分，但提前主動告知 + 簡短說明原因 + 真誠道歉，仍有機會繼續進行。**不要不告而到**，那等於直接退賽。

**Q4：面試前可以吃東西嗎？**
建議吃，避免低血糖。但避免大餐、油膩食物、含洋蔥蒜頭的食物（口氣）、過量含糖飲料（血糖崩盤會更累）。

**Q5：可以禱告 / 冥想 / 做自己的儀式嗎？**
任何讓你穩定的方式都好。心理穩定 > 完美準備。

## 結語

準備不是要把所有問題背起來，而是要讓你在現場「不被任何一題擊倒」。**面試是一場展示自己的對話，不是一場考試**。

**核心原則**：

1. **研究是基本功**：多數人不做，做了就贏。
2. **故事卡是彈藥庫**：5–7 張卡片應對絕大多數題目。
3. **問問題比回答更重要**：好的問題會讓面試官對你印象深刻。
4. **儀式感降低焦慮**：建立你自己的「面試前 SOP」，每次都用同一套。

**行動建議**：到 **面試趣（interview.tw）、求職天眼通、Glassdoor、Dcard 工作版、PTT Tech_Job / Salary 板** 查該公司的真實面試心得，對照你準備好的故事卡，看看還有哪些題目你尚未準備到。把缺口補上，你的面試成功率會明顯提升。
