# 顧問業、金融業、科技業：三大熱門產業全比較

每年大學畢業生最嚮往的三大產業，幾乎都是顧問、金融、科技。它們的共通點是「高薪、高成長、高曝光」，但實際工作內容、文化、生涯路徑卻差異極大。本篇將從產業邏輯、工作日常、薪資、生活型態、職涯路徑等面向，提供完整比較，幫你看見「光環」之外的「真相」。

## 一、產業核心邏輯

| 面向 | 顧問業 | 金融業 | 科技業 |
|------|--------|--------|--------|
| 核心商品 | 解方建議 | 資金與風險管理 | 軟硬體產品 |
| 客戶 | B2B 大企業 | B2B / B2C | B2B / B2C |
| 收入模式 | 專案計費 | 利息、手續費、利差 | 訂閱、廣告、產品銷售 |
| 主要競爭力 | 思考力與簡報力 | 風險判斷與人脈 | 產品力與技術 |
| 工作節奏 | 專案制（6–12 週） | 月度 / 季度循環 | 雙週迭代或每日 release |
| 結果可見度 | 報告 / 簡報 | 帳本 / 績效 | 產品 / 用戶 |

## 二、典型工作內容

### 顧問業
- 拆解客戶問題、做產業研究、製作 PPT 與 Excel 模型、客戶簡報。
- 一週典型節奏：週一 fly-out 到客戶端、週二三四現場工作、週四晚 fly-back、週五在 home office 整理。
- 旅遊頻繁但壓力大。

**一日工作範例（資淺顧問）**：
- 8:30 抵達客戶辦公室
- 9:00–11:00 訪談客戶業務主管
- 11:00–13:00 整理訪談重點、和 senior 對齊
- 13:00–14:00 午餐 + 開會
- 14:00–18:00 做財務模型、寫 PPT
- 18:00–19:00 客戶 daily check-in
- 19:00–22:00 繼續做 deck（在 hotel）
- 22:00–23:00 寫週報給 manager

### 金融業
依職位差異大：
- **投資銀行（IB）**：做併購估值、IPO 規劃、客戶 pitch book。工時極長（80–100 hr/週）。
- **研究員（Equity Research）**：寫產業 / 個股報告、追蹤財報。
- **資產管理（Asset Management）**：管理投資組合、研究公司基本面。
- **商業銀行（Commercial Banking）**：做企業放款、信用評估。

**一日工作範例（資淺 IB Analyst）**：
- 9:00 進辦公室、看 markets、回信
- 10:00–13:00 做 financial modeling
- 13:00–14:00 邊吃邊開會
- 14:00–18:00 修改 pitch book、和 VP 對齊
- 18:00–22:00 客戶會議、做新版本
- 22:00–凌晨 2:00 修改修改再修改
- 凌晨回家睡覺、隔天 9:00 繼續

### 科技業
- **產品經理（PM）**：規劃功能、寫 PRD、開會、分析數據。
- **工程師**：寫程式、Code Review、debug、發布版本。
- **設計師**：做 wireframe、原型、與 PM 工程協作。
- **行銷**：做用戶獲取、廣告、內容、數據分析。
- **營運**：做數據優化、客戶溝通、流程改善。

**一日工作範例（資淺 PM）**：
- 9:30 進辦公室、看數據儀表板
- 10:00–11:00 Sprint Planning
- 11:00–12:00 與設計師討論新功能
- 12:00–13:00 午餐
- 13:00–15:00 寫 PRD
- 15:00–16:00 和工程師 1on1
- 16:00–17:00 用戶訪談
- 17:00–18:30 寫週報、回 Slack
- 18:30 下班

## 三、生活型態比較

| 面向 | 顧問業 | 金融業（IB） | 科技業 |
|------|--------|--------------|--------|
| 平均工時 | 60–80 / 週 | 70–100 / 週 | 40–60 / 週 |
| 出差頻率 | 高（每週） | 中等 | 低 |
| 週末工作 | 偶爾 | 常態 | 較少 |
| 遠端彈性 | 中等 | 低 | 高 |
| 假期 | 14 天 + 彈性 | 14 天但難請 | 14–20 天 |
| 通勤時間 | 視客戶地點 | 進辦公室為主 | 多數可遠端 |
| 工作 / 生活平衡 | 差 | 極差 | 中等到好 |

## 四、薪資與成長

### 顧問業
- **起薪**：中高（約 60K–90K，外商頂尖約 120K+）。
- **獎金**：年終 1–3 個月。
- **成長幅度**：3–5 年可達 PM 等級雙倍薪資。
- **頂尖收入**：Partner 級可達年薪 1,000 萬+。

### 金融業
- **起薪**：最高（IB Analyst 可達 90K–150K + 紅利）。
- **獎金**：佔總收入 30–50%（依市場景氣大幅波動）。
- **成長幅度**：5–10 年可達 VP / MD 等級。
- **頂尖收入**：MD 級可達年薪 2,000 萬+。

### 科技業
- **起薪**：中高（科技大廠 60K–120K，外商可達 150K+）。
- **股票（RSU）**：美國科技大廠（如 Apple、Google、Meta、Amazon、Microsoft 等）股票占總收入 30–50%。
- **成長幅度**：依產品 / 公司規模差異大。
- **頂尖收入**：資深總監 / VP 可達年薪 1,500 萬+。

## 五、適合的人格特質

### 顧問業適合
- 喜歡解問題、邏輯清晰、能在不確定中行動的人。
- 抗壓性強、能適應頻繁出差。
- 善於 PPT、簡報、結構化思考。
- 喜歡多元產業（每 6 週換新產業）。

### 金融業適合
- 對市場、數字敏感。
- 能承受壓力、競爭、長工時。
- 對權力與報酬有強烈動機。
- 擅長人際互動（特別是與高階決策者）。
- 細節控（一個錯字就會被罵）。

### 科技業適合
- 對產品、用戶、創新有熱情。
- 能與多元背景團隊合作。
- 願意持續學習新技術或新工具。
- 喜歡看到「自己做的東西被很多人用」。
- 願意接受變動（產品方向常改）。

## 六、職涯流動性

### 顧問 → 任何產業
因為訓練扎實，是最容易跳槽的職業之一。常見路線：
- 顧問 → 大企業 Strategy / Corp Dev。
- 顧問 → PE / VC（少數）。
- 顧問 → 創業。

### 金融 → PE / VC / Hedge Fund / 企業財務
路線清楚，但跨業難。常見路線：
- IB → PE（最熱門）。
- Equity Research → Hedge Fund。
- IB → 大公司 CFO / FP&A。

### 科技業 → 創業 / 跨產業 PM
靈活度高，特別是 PM 與資料職位。常見路線：
- PM → 新創 PM / Founding Team。
- 工程師 → 全端架構師 / CTO。
- 數據分析 → Data Scientist / 跨產業分析。

## 七、實習階段的差異

### 顧問業
- 暑期實習通常 8–10 週，參與一個完整專案，Return Offer 約 60–80%。
- 常見實習計畫：BCG Summer Associate、Bain ACI、McKinsey Summer Associate。
- 競爭極激烈（錄取率 < 3%）。

### 金融業（IB）
- 超強篩選，10 週實習、加班強度高，Return Offer 是入行門票。
- 常見計畫：Goldman Summer Analyst、Morgan Stanley、JP Morgan。
- 錄取率 < 1%（北美前段）。

### 科技業
- 實習形式多元，從暑期到學期，PM、工程、營運皆有，文化通常較開放。
- 常見計畫：Google STEP、Meta MPK、Microsoft Explore。
- 也有許多本土科技公司（Dcard、KKBOX、iKala）。

## 八、不同產業的「真實隱性挑戰」

### 顧問業的隱性挑戰
- **頻繁旅遊**：個人關係難維持，伴侶關係挑戰大。
- **沒有「自己的產品」**：你做的事情成果都是客戶的。
- **6–8 年後容易厭倦**：相同的工作模式重複。

### 金融業的隱性挑戰
- **健康壓力**：長工時、高壓力，許多 IB 同事在 30 歲就有健康問題。
- **金錢成癮**：高薪會讓你「離不開」，即使你不喜歡工作。
- **狹窄的人際圈**：大部分時間只接觸金融圈的人。

### 科技業的隱性挑戰
- **產品被砍 / 公司倒閉**：你做的東西可能 1 年後消失。
- **持續學習壓力**：3 年前的技術現在沒用。
- **裁員風險**：大廠 layoff 越來越頻繁。

## 九、選擇建議

問自己三個問題：

### 1. 我願意付出多少時間在工作上？
不一樣的產業有不一樣的「工時 × 強度」比例：
- 顧問：高工時 + 中強度。
- 金融：極高工時 + 高強度。
- 科技：中工時 + 高強度。

### 2. 我喜歡解別人的問題，還是創造自己的東西？
- 顧問偏前者。
- 科技偏後者。
- 金融兩者皆有。

### 3. 我重視什麼樣的成長？
- 顧問：廣度（多產業 × 多問題類型）。
- 金融：收入（年薪曲線最陡）。
- 科技：產品力（看著產品成長）。

## 十、實習通的真實對照

到實習通搜尋這三個產業的心得時，你會發現：
- **顧問業**心得：常出現「pressure cooker」、「很有成就感但會 burnout」、「學習速度快」。
- **金融業**心得：常出現「凌晨還在修 deck」、「team culture 很重要」、「客戶的 ego 很大」。
- **科技業**心得：常出現「產品方向常改」、「跨部門溝通難」、「主管影響很大」。

每個產業都有人活得閃亮、也都有人後悔。**你的選擇 × 你的努力**，才是真正的答案。

## 十一、進階：「混合路徑」的可能性

不一定要選一個從一而終。常見的混合路徑：

### 路徑 A：顧問 → 科技 PM
顧問 3–5 年累積商業判斷力，跳到科技業擔任 PM。

### 路徑 B：金融 → 創投
IB 5 年累積估值能力，跳到 VC 投資新創。

### 路徑 C：科技工程師 → 創業
工程師 5 年累積技術能力，創立自己的公司。

### 路徑 D：顧問 → 創業 → 企業高管
跨多領域累積，最終成為企業領導人。

## 十二、FAQ

**Q1：三個產業薪資差很多嗎？**
起薪差距：金融 > 顧問 ≈ 科技。長期：科技（含 RSU） > 金融 ≈ 顧問。

**Q2：哪個產業最適合非商管 / 非資工背景？**
顧問業歡迎多元背景。金融業偏好商管 / 經濟。科技業 PM 也歡迎跨領域。

**Q3：女性在這三個產業哪個最友善？**
科技業相對較友善（彈性、文化開放）。金融業女性比例最低，特別是 IB。

**Q4：年紀大才轉行還來得及嗎？**
顧問業歡迎 MBA 後進入。金融業偏好年輕進入。科技業最歡迎跨業者。

**Q5：可以先做一個再轉嗎？**
可以。許多人「顧問 5 年 → 科技」是經典路徑。

## 結語

沒有最好的產業，只有最適合你的產業。

**核心原則**：
1. **看真實生活，不是品牌光環**：到實習通看真實心得。
2. **試做勝於想像**：實習是最便宜的試做。
3. **接受 trade-off**：每個產業都有代價。
4. **不要被「最好」綁架**：你的「最好」是你能持續享受的。

**行動建議**：到實習通查看「顧問」、「金融」、「科技」三個關鍵字的真實心得，比較工作內容、文化與工時，找出與你價值觀最貼近的那一個。挑出 1–2 個感興趣的方向，做一段對應的實習試水溫。
