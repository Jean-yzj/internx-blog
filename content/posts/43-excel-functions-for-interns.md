# Excel / Google Sheets 進階：實習生最常用的 10 個函數

「我會 Excel 啊，可以做樞紐分析。」——這句話幾乎是每個實習生面試時都會講的標準答案。但真正進公司後你會發現，主管丟給你一份 5,000 筆的客戶名單，要你「比對最近一週下單的人，並把客單價超過 1,000 元的圈出來」，這時候你才驚覺自己不會 VLOOKUP，更不知道什麼是 SUMIFS。本篇文章整理實習生最常被要求的 10 個函數，每一個都附上行銷、財務、HR 場景的實際用法，幫你在實習第一週就能交出能看的報表。

## 一、為什麼函數比樞紐分析表更重要？

很多人以為「會樞紐分析」就等於會 Excel，但實際在公司裡，**樞紐分析是「呈現結果」的工具，函數才是「處理資料」的工具**。一份雜亂的原始資料如果沒有先用函數清洗、串接、計算，樞紐分析做出來只會是一堆錯的數字。

### 函數的三個層級
- **基礎層**：SUM、AVERAGE、COUNT。這個層級不算會 Excel，只算識字。
- **應用層**：VLOOKUP、IF、SUMIFS。實習生最低門檻，會這層才能交付一般報表。
- **進階層**：INDEX-MATCH、ARRAYFORMULA、FILTER。能寫出這層的人，往往可以把別人需要好幾天的工作壓縮到一個下午完成。

本篇要教的，是讓你從基礎層直接跳到應用層、甚至摸到進階層的 10 個關鍵函數。

## 二、查找與比對類：VLOOKUP / XLOOKUP / INDEX-MATCH

這是實習生面試最常被問的考點，也是最常用的場景。

### 1. VLOOKUP：經典中的經典
語法：

```
=VLOOKUP(查找值, 查找範圍, 回傳的欄位編號, FALSE)
```

**行銷情境**：你有一份「廣告投放成效」表，另一份是「商品 SKU 與品類對照」。你想知道每個 SKU 是哪個品類，就用 VLOOKUP 把品類欄位拉過來。

**注意限制**：VLOOKUP 只能「向右查找」，查找值必須在查找範圍的第一欄。所以才有了下面的進階版本。

### 2. XLOOKUP：VLOOKUP 的繼承者（推薦）
語法：

```
=XLOOKUP(查找值, 查找欄, 回傳欄, [找不到時回傳], [比對模式], [搜尋模式])
```

**HR 情境**：要查某員工編號對應的「主管姓名」，但員工編號在第 5 欄、主管姓名在第 2 欄。VLOOKUP 做不到，XLOOKUP 一行解決。Google Sheets 也已經支援。

### 3. INDEX-MATCH：傳統 Excel 必備組合
語法：

```
=INDEX(回傳範圍, MATCH(查找值, 查找範圍, 0))
```

**財務情境**：建立財務模型時，常需要從一個「年度假設表」中根據「年份 + 科目」雙條件查找。INDEX-MATCH 的彈性遠勝 VLOOKUP，是顧問業、投資銀行最愛的組合。

## 三、條件加總與計數：SUMIFS / COUNTIFS

### 4. SUMIFS：多條件加總
語法：

```
=SUMIFS(加總範圍, 條件範圍1, 條件1, 條件範圍2, 條件2, ...)
```

**行銷情境**：你想算「2026 年 4 月、廣告平台是 Meta、且活動類型是 Conversion」的總花費，一個 SUMIFS 就解決。手動篩選再加總是新手做法，老手只寫公式。

### 5. COUNTIFS：多條件計數
**HR 情境**：算「年資 3 年以上、職等 P5 以上、女性員工」的人數。COUNTIFS 一行公式比手動篩選快非常多，而且可以隨資料更新自動重算。

## 四、邏輯判斷：IF / IFS

### 6. IF / IFS：條件分歧
語法：

```
=IF(條件, 為真時回傳, 為否時回傳)
=IFS(條件1, 結果1, 條件2, 結果2, ..., TRUE, 預設結果)
```

**業務情境**：要把客戶分群為「VIP / 一般 / 新客」，依照消費金額判斷：

```
=IFS(A2>=10000, "VIP", A2>=3000, "一般", TRUE, "新客")
```

實習生常見錯誤是把 IF 嵌套五六層，可讀性極差。**請改用 IFS，整潔且容易維護**。

## 五、文字處理：TEXT / LEFT / RIGHT / MID

### 7. TEXT：把數字格式化
語法：

```
=TEXT(值, "格式")
```

**行銷情境**：日期欄位是 `2026/05/08`，你想顯示成「2026 年 5 月」用在報表標題：

```
=TEXT(A2, "yyyy 年 m 月")
```

或是把金額顯示成「NT$ 1,234」：

```
=TEXT(B2, "NT$ #,##0")
```

### 8. LEFT / RIGHT / MID：字串切割
**HR 情境**：員工編號是 `TW-2024-0521-MKT`，你想抽出「年份」做樞紐分析：

```
=MID(A2, 4, 4)
```

抽出「部門代碼」：

```
=RIGHT(A2, 3)
```

切記：抽完之後若要當數字使用，要再用 `VALUE()` 包一層轉型。

## 六、日期函數：實習生天天會用

### 9. 日期函數家族
最常用的四個：

| 函數 | 用途 | 範例 |
|------|------|------|
| TODAY() | 回傳今天日期 | =TODAY() |
| DATEDIF() | 計算兩日期差距 | =DATEDIF(A2, TODAY(), "Y") |
| EOMONTH() | 月底日期 | =EOMONTH(A2, 0) |
| WEEKDAY() | 星期幾（1-7） | =WEEKDAY(A2, 2) |

**HR 情境**：算員工年資，用 DATEDIF。
**財務情境**：算應收帳款帳齡（天數），用 `=TODAY() - 出貨日`。
**行銷情境**：把活動依「星期幾」分組分析點擊率，用 WEEKDAY。

## 七、Google Sheets 限定的神級函數

### 10. FILTER 與 ARRAYFORMULA
這兩個在 Excel 365 也有，但 Google Sheets 上更常用，因為實習公司大多用 Google Workspace。

**FILTER 語法**：

```
=FILTER(資料範圍, 條件1, 條件2, ...)
```

**行銷情境**：從一份 5,000 筆訂單中，動態抓出「金額 > 1000 且地區是台北」的所有訂單，自動更新：

```
=FILTER(A2:E5000, B2:B5000>1000, C2:C5000="台北")
```

**ARRAYFORMULA 語法**：

```
=ARRAYFORMULA(對整欄的運算式)
```

它能讓你**只寫一條公式，自動套用整欄**，不用往下拉。維護報表時超方便，公司資料新增時自動延伸。

## 八、樞紐分析表：函數之外的最強搭檔

雖然樞紐不是函數，但每個實習生都該會。三個必懂操作：

### 1. 行/列/值/篩選四個欄位的拖放邏輯
- 行：你想看的「分組維度」，例如「品類」
- 值：要被加總或計數的欄位，例如「銷售額」
- 篩選：頂端的下拉條件，例如「只看 2026 Q1」

### 2. 「值欄位設定」改成百分比
銷售額拖到值欄位後，預設是加總。點開「值欄位設定」可改為「佔總和的百分比」、「佔列總計的百分比」，是行銷做活動成效拆解最實用的功能。

### 3. 樞紐分析表的「重新整理」
原始資料更新後，樞紐不會自動同步。要記得右鍵「重新整理」，這是新手最常忘記的步驟，導致報表上的數字是錯的。

## 九、常見錯誤與除錯心法

### 1. `#N/A` 錯誤
99% 是查找函數找不到匹配值。檢查：
- 兩邊是否都有空白字元（用 `TRIM` 清理）
- 數字 vs 文字格式（一個是 "123"，一個是 123）
- 大小寫差異（VLOOKUP 是不分大小寫的，但有些情境很微妙）

### 2. `#REF!` 錯誤
範圍被刪除或公式拖移到無效區域。改用「絕對參照」`$A$1` 鎖定範圍。

### 3. 公式跑很慢
通常是 VLOOKUP 對整欄 `A:A` 操作。改成具體範圍 `A2:A5000` 速度會快 10 倍。

## 十、FAQ：實習生問函數常見問題

**Q1：我都用 VLOOKUP，需要改學 XLOOKUP 嗎？**
建議改。XLOOKUP 寫法更直觀、支援雙向查找、可以指定找不到時的回傳值。新公司的範本越來越多用 XLOOKUP。

**Q2：函數寫了一堆，看不懂自己在寫什麼怎麼辦？**
拆成「輔助欄」。先用 D 欄算出查找鍵、E 欄做比對、F 欄做最終運算。**讀得懂的公式比一行寫到底更專業**。

**Q3：要學 VBA 或 Apps Script 嗎？**
實習階段先把上述 10 個函數練熟。等你發現「同樣的步驟每週重做一次」時，再學 Apps Script 自動化。一開始就學 VBA 通常會卡住而失去動力。

**Q4：Excel 跟 Google Sheets 哪個好？**
看公司。新創、行銷、商務職多用 Google Sheets；金融、製造、傳產多用 Excel。**兩個都會用是最大公約數**，函數 90% 共通。

## 十一、結語：函數是實習生最被低估的硬實力

很多實習生把時間花在學「PowerPoint 設計」「Notion 美化」，卻忽略 Excel 函數。但企業真正願意付薪水的，是「能把 5,000 筆資料變成 1 頁洞察」的能力，而那靠的就是這些函數。

**三個關鍵原則**：
1. **先學會 10 個就夠**：不要一次想學 50 個，把上面這 10 個練到反射動作就贏 80% 的同學。
2. **每個函數都有「實際情境」才記得住**：硬背語法會忘，但「比對 SKU 品類」「分群客戶 VIP」這種場景會記一輩子。
3. **會寫不等於寫得好**：可讀性、效能、維護性，是讓你的報表被主管採信的關鍵。

**行動建議**：到實習通搜尋你目標的產業與職位，看看實習心得中提到「每天用 Excel 處理什麼資料」，再回頭把對應的函數練一遍。下週進公司時，你的報表會讓主管多看你兩眼。
