# 實習轉正攻略：如何把暑期實習變成 Return Offer

對許多大學生來說，**「Return Offer」（實習結束後拿到正職邀請）是大三暑假的終極目標**。一張 Return Offer 不只是工作保障，更是你大四能心無旁騖完成學業、不再投履歷的關鍵。本篇將教你如何在實習階段最大化轉正機率，並補上台灣本土具備結構性 return offer 機制的公司（台積電 DNA、星展、中信、玉山等）對照。

> **使用本文前請注意**：以下提到的各家公司實習計畫名稱、報名時程、Return Offer 機制、起薪、簽約金均逐年變動，請以該屆官方網站當期公告為準。

## 一、Return Offer 的真相

### 1. 它不是「乖乖把任務做完就有」

**部分公司有事實上的 return offer 比例上限**（依年度 HC 而定），代表並非每位實習生都會拿到 offer。台灣外商科技業、投資銀行、顧問業特別明顯。

### 2. 它是「主管推薦 + HR 程序」的雙重結果

**主管推薦通常是最大關鍵，HR 流程與名額為次要條件**（具體權重依公司而定）。你能不能拿到 offer，第一線決策者通常是直屬主管。

### 3. 它通常在實習結束前 2–4 週決定

最後幾週的表現比前幾週更關鍵。多數結構性實習計畫（MBB、IB、大型科技外商、台積電 DNA、星展、中信）都在實習中後段進行 final evaluation。

### 4. Return Offer 的等級也會被決定

**此分級為示意**——部分外商 IB／MBB 會有 strong offer / standard offer / no offer 之內部差異化（影響起薪、組別選擇、未來晉升）；台灣本土公司多半為「有 offer／無 offer」兩種。

- **Strong Offer**：高薪、好職位、可選團隊（多見於外商 IB／顧問業頂尖實習生）。
- **Standard Offer**：標準正職邀請。
- **Conditional / B Offer**：有 Offer 但條件較差（如延後報到、特定 rotation）。
- **No Offer**。

你的目標應該是 Strong / Standard Offer。

### 5. 不同產業的 Return Offer 比例（業界普遍引用區間）

> **重要提醒**：以下為**美國／全球業界普遍引用區間**。台灣本土公司、外商台灣分公司 Return Offer 機制差異大，台灣本土 IB／MBB 機會極少（多半透過 HK／SG office 招募），請勿直接套用以下美國比例至台灣本地實習。

| 產業 | Return Offer 比例（美國／全球） | 台灣本地對照 |
| --- | --- | --- |
| 顧問業（MBB：McKinsey、BCG、Bain） | 70–95%（近年已取消「實習自動 offer」政策） | 台灣 MBB 實習名額極少（每屆個位數），多半透過 HK／SG 招募 |
| 投資銀行（美國 bulge bracket：GS / MS / JPM 等） | 60–80%（拿到通常會 take） | 台灣本地 IBD 機會極少，多半透過 HK／SG office；本土券商（元大、凱基、群益）IBD 規模較小 |
| 大型科技外商（Google、Meta、Microsoft、Amazon） | 50–80%（依年度與分公司差異大） | 台灣分公司實習名額少、return offer 機制依當年人力規劃而定 |
| 本土科技／新創 | 無公開統計，差異極大 | **台積電 DNA Internship 設「Advance Offer」機制**；新創多半個案／開缺式轉正 |
| 廣告／媒體 | 普遍偏低、依公司與年度而定 | 本土代理商（奧美、李奧貝納、台灣電通）多半專案制、HC 緊縮 |

### 6. 台灣本土結構性 Return Offer 機制（具名）

對台灣大學生而言，比起搬美國數據更重要的是**台灣本土具備「結構性」（非個案、有設計）return offer 機制**的公司：

#### 台積電（TSMC）DNA Internship

- 計畫設計：**Develop（培訓）+ Navigate（專案執行）+ Advance Offer（提早給正職邀請）** 三大內涵。
- 對象：大三以上、優先給 2026 屆碩博士、理工＋商管領域。
- 機制：**表現優異的實習生可在實習結束前後獲得 Advance Offer Letter（含研發替代役職缺）**，等同提早卡位正職。
- 2025 年時程示意：5/5 報名截止、面試至 6/20、實習 7/7–8/31。
- 報名請以台積電 careers 官網與各校（台清交成）聯合研發中心當期公告為準。

#### 星展（DBS Taiwan）暑期實習

- **雙軌制（為期 8 週）**：
  - **專案式實習（Structured Internship）**：專為**最後一學年學生**設計，含培訓平台、海外見習機會。
  - **一般暑期實習（General Internship）**：開放各年級學士／碩士。
- 轉正機制：實習結束前經主管評核，**通過者有機會於隔年七月獲「星展儲備幹部」或正職職位**。

#### 中國信託（CTBC）暑期實習

- 採**階段式轉正設計**（暑期實習 → 學期實習 → 正職），非一次性 return offer。
- 申請門檻：大三以上在學（含研究所），GPA 3.7 / 84 分以上、TOEIC 800 或 JLPT N2。

#### 其他金融 MA（Management Associate 儲備幹部）相關計畫

- 玉山銀行、國泰金控、富邦金控、台新銀行等本土金控普遍有結構性實習＋ MA 計畫，部分計畫對暑期實習表現優異者直接內推 MA 流程；具體機制依該行當期招募公告為準。

#### 本土科技業具名實習計畫

- 聯發科 IC Camp、廣達 IC Camp、緯創、華碩、宏碁等皆有結構性暑期實習計畫，部分計畫對表現優異者直接邀請畢業後加入研發替代役或正職。

## 二、實習第一週：定錨期

### 1. 與主管做「期待對齊」

第一次 1on1 必問三題：

#### Q1：「您希望我在實習結束時達到什麼？」

這個問題能讓主管把抽象期待具體化。

#### Q2：「您過去帶過最優秀的實習生有什麼共通點？」

這是「成功公式」，主管會親自告訴你。

#### Q3：「我有哪些表現是您會認為『超出預期』的？」

直接問如何拿 strong offer，主管通常會給你 2–3 個明確指標。

> **台灣本土公司文化差異提醒**：外商台灣分公司（Google / Microsoft / DBS 等）可直接套用上述 1on1 期待對齊 SOP；台灣本土公司／傳統產業主管多半較不熟悉「1on1」、「期待對齊」之 US tech 用語，建議改為較含蓄的表達：「請主管告訴我這幾個月希望我達成什麼？我會回去整理一份計畫並再跟您 confirm。」

把答案寫下來，這是你 8 週的北極星。

### 2. 認識團隊每一個人

不要只認識自己組員。**請各部門前輩喝杯咖啡**，建立未來幫你說話的「支持者網絡」。

#### 第一週的「咖啡 marathon」

- 直屬主管。
- 團隊內所有同事。
- 跨部門的 1–2 位關鍵 stakeholder（關鍵利害關係人）。
- 公司內的 1 位資深前輩（mentor 角色）。

每場 15–20 分鐘，問同一組問題：

- 你的角色是什麼？
- 你最近在忙什麼？
- 我能為你做什麼？

這個動作會讓你「被看見」。

### 3. 建立筆記系統

從第一天就用 Notion 或 OneNote 記錄：

- 每個會議的決策。
- 每個專業詞彙的解釋。
- 每個流程的 SOP（Standard Operating Procedure 標準作業流程）。

#### 範本

- 「會議紀錄」資料夾：每場會議一個檔。
- 「公司詞彙表」：用內部術語的清單。
- 「我的 to-do」：每天 5–10 個小事項。
- 「學習筆記」：學到的觀念、書、資源。

這份筆記，會在實習中後期成為你的「超能力」。

## 三、實習中期（第 3–6 週）：價值期

### 1. 找一個「Quick Win」

在第 4 週前完成一件對團隊有貢獻的小事。例如：

- 自動化一個重複的 Excel 流程。
- 整理一份過去沒人整理的 SOP。
- 主動寫一份競品分析。
- 為團隊知識庫建立一個 Notion 主題頁。
- 找出某流程的 bottleneck 並提出解方。

這會讓主管發現「你和其他實習生不一樣」。

#### 真實情境舉例

一位在某金融科技新創實習的政大商學院 A 同學發現團隊每週要花 3 小時手動整理「上週用戶回饋」。她用 Google Apps Script 寫了一個自動化腳本，每週節省 3 小時 × 4 人 = 12 工時。

主管在週會公開稱讚：「A 解決了一個我們忍受了 6 個月的問題。」

她最後的 Return Offer 條件是該屆最好的之一（含 sign-on bonus 與選團隊權）。

### 2. 主動承擔超出範圍的任務

當你聽到主管說「啊這個沒人做」、「這個拖很久了」——立刻說「我可以試試」。

#### 主動承擔的範例

- 「沒人寫的競品分析」→ 我可以做。
- 「沒人整理的會議紀錄」→ 我可以接。
- 「沒人 review 的 PRD（Product Requirements Document 產品需求文件）」→ 我可以幫。

### 3. 每週寄「週報」給主管

即使主管沒要求，每週五寄一封：

#### 週報結構

```
Subject: [實習生週報] Week 3 - 進度與下週計畫

Hi 主管，

本週完成：
- [專案 A] 完成 X 部分，預計下週進入 Y 階段。
- [專案 B] 與設計師確認 wireframe（介面草圖），今晚進入開發。
- [其他] 整理出競品分析草稿（連結附上）。

本週學到：
- 學會 X 工具的進階用法。
- 從 Y 同事身上學到 Z 觀念。

下週計畫：
- 完成 [專案 A] 的 Y 階段。
- 開始 [專案 C]。

需要的支援：
- 是否方便週二一起 review [專案 A]？
- 對於 [某個決策] 我有 2 個想法，希望和您討論。

謝謝！
ZZ
```

這種主動回報是 Return Offer 的隱藏神器。

### 4. 建立跨部門的「支持者」

不要只當「直屬主管的實習生」，要當「整個部門的好助手」。

#### 做法

- 主動幫設計師整理用戶資料。
- 主動為工程師準備 PRD 範本。
- 主動為業務蒐集競品資訊。

這樣 Return Offer 評選時，**會有 5 個人為你說話，不是只有 1 個**。

## 四、實習後期（第 7–10 週）：證明期

### 1. 完成一個「可被講述」的成果

最後一週前，你要有一件「能在面試中講 5 分鐘」的事。

#### 標準

- 你是主導者（不是助手）。
- 有量化成果。
- 對團隊 / 公司有實質影響。

### 2. 主動約主管「中期回饋」

「我這週剛好實習過半，想請您給我一些回饋，特別是哪些地方需要改進。」

**這展現的不是脆弱，是成長心態**。

#### 中期回饋對話

> 你：「我想請您給我一些回饋，特別是您覺得我哪些地方做得好、哪些需要改進。」
> 主管：「你做得很好，但 X 部分可以再加強。」
> 你：「謝謝您的回饋。針對 X，我打算 Y 來改進。下週我會再回報進度。」

這場對話結束後，你已經在主管心裡留下「會反思、會行動、會跟進」的印象。

### 3. 與 HR 主動溝通

不要等 HR 來找你，最後 2–3 週可以禮貌詢問：

> 「請問貴公司今年是否有 Return Offer 名額？需要我做什麼準備？」

對於台積電 DNA、星展、中信等具備結構性 return offer 機制者，可直接詢問該年度 Advance Offer / 儲備幹部 / 學期實習轉正之時程與決策節點。

### 4. 寫實習結束的「成果簡報」

製作一份完整的「實習成果簡報」（10–15 張投影片）：

- 我做了什麼。
- 帶來什麼影響。
- 學到什麼。
- 接下來的建議。

這份簡報會留在公司，成為決定 Return Offer 的關鍵之一。

## 五、影響 Return Offer 的關鍵指標

### 1. 工作表現

- 任務交付速度與品質。
- 主動性與創造力。
- 跨部門協作能力。

### 2. 文化契合度

- 和團隊互動是否自然。
- 是否認同公司價值觀。
- 是否願意長期投入。

### 3. 商業判斷力

- 能否從用戶 / 客戶角度思考。
- 能否平衡短期與長期。
- 能否在資源限制下做決策。

### 4. 學習力

- 能否快速吸收新工具。
- 能否從錯誤中調整。
- 能否主動找資源。

### 5. 影響力

- 是否帶來具體的改變。
- 是否影響到其他同事的工作方式。
- 是否被「記住」。

## 六、實習中容易踩的五大地雷

### 地雷 1：過度被動

等到主管來分配任務，從不主動。

**徵兆**：

- 每天等 task 來。
- 不問問題。
- 沒主動約 1on1。

**矯正**：每週主動找主管 1 次「我能不能做 X」的對話。

### 地雷 2：不問問題或問太多沒準備的問題

兩種極端都不好。

**正確做法**：

- 問問題前先 Google / 看內部文件。
- 把問題整理成清單，1on1 時集中問。
- 問「策略問題」（為什麼）多於「執行問題」（怎麼做）。

### 地雷 3：不建立人脈

只和直屬主管聊天，其他人完全不熟。

**矯正**：每週至少和 1–2 位非直屬同事吃午餐或喝咖啡。

### 地雷 4：太快談薪水

第一週就問 Return Offer 條件。

**矯正**：聚焦在做事，到實習剩 2–3 週時再禮貌詢問。

### 地雷 5：最後幾週鬆懈

以為要結束了就少做事，這正是評分期。

**矯正**：把最後 2 週當成「決勝期」，比第一週更認真。

## 七、就算沒拿到 Return Offer，也別氣餒

不少優秀的學生因為 HR 名額限制、組織重組等因素未獲 Return Offer，但他們的下一步：

- 用實習主管當推薦人。
- 把實習經驗精準寫進履歷。
- 兩個月內拿到其他公司 Offer。

> **沒有 Return Offer ≠ 表現不好**，但有 Return Offer ≈ 你已經是準正職員工。

### 沒拿到 Return Offer 的應對策略

#### 1. 主動詢問原因

> 「謝謝您過去幾個月的指導。如果方便，我希望了解具體哪些方面我可以再加強，以便未來改進。」

#### 2. 請求轉介紹

> 「您是否認識其他公司有類似職缺，方便為我介紹嗎？」

#### 3. 維繫關係

即使沒 Offer，主管也是未來的推薦人。每 3–6 個月更新近況。

#### 4. 善用台灣本地求職管道補位

具名管道：

- **求職平台**：實習通、CakeResume、Yourator、104、1111
- **社群**：Dcard 工作板、Facebook 求職社團、LinkedIn（請主管或同事公開 endorse / 寫推薦短評）
- **學校資源**：學校職涯中心、校友會、職涯講座
- **加速器／訓練營**：AAMA 台北搖籃計畫、AppWorks School、ALPHA Camp、Garage+（時代基金會）

## 八、Return Offer 後的進階談判

拿到 Return Offer 不是終點。你可以談：

### 1. 起薪

若你在業界已有比較基準（其他 Offer），可以嘗試爭取起薪上修。**台灣本土公司（如金融 MA、台積電）多半有固定起薪 band，談判空間較小**；外商相對有彈性，但仍依公司與職等而定。

### 2. 部門 / 團隊輪調

有些大公司允許你在簽約後選擇進入哪個團隊。台積電 DNA Advance Offer 多半會與你的研發替代役職缺一同確認；金融 MA 多半有指定 rotation 計畫。

### 3. 報到日期彈性

若想畢業後出國旅遊、休息或服役，可以爭取延後 1–2 個月（特別是男性同學的兵役因素）。

### 4. Sign-on Bonus

**部分頂尖外商科技業／金融業對 entry-level 提供 sign-on bonus**（常見 NT$10–30 萬區間，依職等與公司差異大）；台灣本土公司多半沒有此項目；若你有競爭性 offer 較容易爭取。

### 5. 升遷時程承諾

能否討論第一次績效檢視時間點。

## 九、各階段自我檢核

### 第 4 週自問

- 我和主管的 1on1 都被善用嗎？
- 我認識的同事超過 5 位嗎？
- 我有 1 個「即將完成」的具體成果嗎？

### 第 6 週自問

- 我是否有主動承擔的紀錄？
- 我是否有跨部門的合作？
- 我是否被同事「記住」（不是只是「實習生」這個角色）？

### 第 8 週自問

- 我能用 5 分鐘講清楚我這次實習的成果嗎？
- 我有 3 個能寫進履歷的量化結果嗎？
- 我有 2 位以上願意幫我寫推薦信的人嗎？

## 十、Return Offer 的「跨產業差異」（含台灣本地對照）

### 顧問業

- **美國 MBB**：return offer 70–95%，但已取消「實習自動 offer」政策、改為 performance-based。
- **台灣／大中華區 MBB**：實習名額極少，多半透過 HK／SG office 招募，每屆個位數。
- **台灣本土顧問**：四大會計師事務所之顧問部門（PwC Consulting、Deloitte Consulting、KPMG Advisory、EY Consulting）有自己的實習計畫；本土策略顧問（如博晟、宇柏）規模較小。
- 評估標準：思考力、簡報力、客戶溝通。
- Return Offer 多半 8 週後立刻決定。

### 投資銀行

- **美國 bulge bracket**：return offer 60–80%，「拿到 = 入行門票」。
- **台灣本地 IBD**：機會極少，多半透過 HK／SG office；本土券商（元大、凱基、群益、永豐金）IBD 規模較小。
- 評估標準：細節控、抗壓性、團隊配合。

### 科技業

- **外商台灣分公司**（Google 台灣、Microsoft 台灣、AWS 台灣、Meta 台灣等）：實習名額少、return offer 機制依當年人力規劃。
- **本土科技**：
  - 台積電 DNA Internship（Advance Offer 機制 + 研發替代役）。
  - 聯發科 IC Camp、廣達、緯創、華碩、宏碁等本土科技業實習。
  - Dcard、Appier、KKBOX、91APP、Pinkoi、iKala、KKday 等新創多半個案／開缺式轉正。
- 評估標準：產品思維、跨部門協作、學習力。
- 部分公司 Return Offer 可延後決定（畢業前才確認）。

### 金融業

- 星展（DBS）雙軌 8 週、最後一學年學生有「儲備幹部」轉正路徑。
- 中國信託：階段式（暑期實習 → 學期實習 → 正職）。
- 玉山、國泰、富邦、台新、第一金、兆豐等本土金控普遍有結構性實習＋ MA 計畫。
- 評估標準：穩定度、細節、流程理解、雙語能力。

### 廣告／媒體

- 本土代理商（奧美、李奧貝納、台灣電通、群邑、達彼思）多半專案制、HC 緊縮。
- 評估標準：創意執行、客戶服務。
- Return Offer 比例普遍偏低，需要主動爭取。

## 十一、FAQ

**Q1：實習表現很好但公司沒 Return Offer 名額怎麼辦？**
請主管寫推薦信，並主動詢問是否能介紹其他公司。

**Q2：可以同時申請多家 Return Offer 嗎？**
法律上沒有禁止。**多數正式 return offer 會要求一定期限內回覆**（外商 IB／顧問業常 1–4 週 exploding offer），台灣本土公司期限相對較寬鬆；同時申請其他 offer 並未被禁止，但需注意各家回覆時程與道德問題。

**Q3：Return Offer 一定要接受嗎？**
不一定。你可以拒絕（特別是發現公司不符你期待）。男性同學另需考量兵役時程。

**Q4：拿到 Return Offer 還能找其他工作嗎？**
看合約。多數允許，但時間有限制。

**Q5：實習表現一般可以拿 Return Offer 嗎？**
看公司。台積電 DNA 的 Advance Offer 為「擇優」（非全給），星展、中信、玉山等本土金融則依該年 HC 與評核結果而定。部分外商有「保底名額」，但好的職位、團隊、薪資只給頂尖實習生。

**Q6：男性同學的兵役（研發替代役、一般役）如何影響 Return Offer 報到時間？**
- **研發替代役**：台積電 DNA Advance Offer 多半直接連動研發替代役職缺，等同提早卡位正職＋兵役。聯發科、廣達等本土科技業亦有研替名額。
- **一般役 / 軍訓役**：部分本土公司（金融、傳產）接受 offer 後等待 4 個月–1 年服役再報到；部分外商接受度依該年 HC 而定，建議在 offer 階段直接詢問 HR 報到日期彈性。

## 十二、提升 Return Offer 機率的「終極清單」

實習開始前：

- 認識公司的 3 個目標客戶。
- 讀公司過去 6 個月的新聞。
- 列出 5 個我想學到的能力。
- **若是台積電 DNA、星展、中信、玉山等具備結構性 return offer 機制的計畫，提早了解該計畫的轉正設計**（advance offer letter / 儲備幹部 / 階段式實習）。

第 1 週：

- 與主管做期待對齊。
- 認識所有團隊成員。
- 建立筆記系統。

第 2–4 週：

- 完成第一個 Quick Win。
- 開始每週寄週報。
- 至少跨部門合作 1 次。

第 5–6 週（中期）：

- 主動約主管 mid-term feedback。
- 確認自己的「故事」開始成形。
- 至少和 5 位同事建立關係。

第 7–8 週（後期）：

- 完成可講述的核心成果。
- 主動詢問 Return Offer 流程。
- 開始準備成果簡報。

第 9–10 週：

- 進行成果簡報。
- 確認 Return Offer 結果。
- 寫感謝信給所有合作過的同事。

## 結語

Return Offer 不是運氣，而是「做對的事 × 持續做」的結果。當你把實習當成「為期 10 週的真實工作試用」，你的態度會自然變得不同。

**核心原則**：

1. **第一週定錨**：問清楚「成功是什麼樣子」。
2. **持續主動**：每週都比上週做得多一點。
3. **建立支持者網絡**：不只討好主管，要討好整個團隊。
4. **最後幾週是決勝期**：不要鬆懈，要全力衝刺。

**行動建議**：

- 到實習通查找你目標公司的「轉正比例」與「轉正關鍵」相關真實心得，提早準備你的轉正策略。
- 若目標為**台積電 DNA、星展、中信、玉山**等具結構性 return offer 機制者，提早了解該計畫的具體設計（advance offer / 儲備幹部 / 階段式實習），這比泛泛討論 US bulge bracket 比例更實用。
- 把這份檢核清單列印出來，貼在你的書桌前，每週對照進度。
