# 實習轉正攻略：如何把暑期實習變成 Return Offer

對許多大學生來說，**「Return Offer」（實習結束後拿到正職邀請）是大三暑假的終極目標**。一張 Return Offer 不只是工作保障，更是你大四能心無旁騖完成學業、不再投履歷的關鍵。本篇將教你如何在實習階段最大化轉正機率。

## 一、Return Offer 的真相

### 1. 它不是「乖乖把任務做完就有」
許多公司明確規定 Return Offer 比例（如 60%、80%），代表**有人會被淘汰**。

### 2. 它是「主管推薦 + HR 程序」的雙重結果
你的直屬主管的支持是 70%，HR 流程與名額是 30%。

### 3. 它通常在實習結束前 2–4 週決定
最後幾週的表現比前幾週更關鍵。

### 4. Return Offer 的等級也會被決定
- A+ Return Offer：高薪、好職位、可選團隊。
- A Return Offer：標準。
- B Return Offer：有 Offer 但條件較差。
- 沒 Offer。

你的目標是 A+。

### 5. 不同產業的 Return Offer 比例

| 產業 | Return Offer 比例 |
|------|---------------------|
| 顧問業（MBB） | 70–90% |
| 投資銀行 | 80%（拿到的人通常會 take） |
| 大型科技外商 | 60–80% |
| 本土科技 / 新創 | 50–70% |
| 廣告 / 媒體 | 30–60% |

## 二、實習第一週：定錨期

### 1. 與主管做「期待對齊」
第一次 1on1 必問三題：

#### Q1：「您希望我在實習結束時達到什麼？」
這個問題能讓主管把抽象期待具體化。

#### Q2：「您過去帶過最優秀的實習生有什麼共通點？」
這是「成功公式」，主管會親自告訴你。

#### Q3：「我有哪些表現是您會認為『超出預期』的？」
直接問如何拿 A+，主管通常會給你 2–3 個明確指標。

把答案寫下來，這是你 8 週的北極星。

### 2. 認識團隊每一個人
不要只認識自己組員。**請各部門前輩喝杯咖啡**，建立未來幫你說話的「支持者網絡」。

#### 第一週的「咖啡 marathon」
- 直屬主管。
- 團隊內所有同事。
- 跨部門的 1–2 位關鍵 stakeholder。
- 公司內的 1 位資深前輩（mentor 角色）。

每場 15–20 分鐘，問同一組問題：
- 你的角色是什麼？
- 你最近在忙什麼？
- 我能為你做什麼？

這個動作會讓你「被看見」。

### 3. 建立筆記系統
從第一天就用 Notion 或 OneNote 記錄：
- 每個會議的決策。
- 每個專業詞彙的解釋。
- 每個流程的 SOP。

#### 範本
- 「會議紀錄」資料夾：每場會議一個檔。
- 「公司詞彙表」：用內部術語的清單。
- 「我的 to-do」：每天 5–10 個小事項。
- 「學習筆記」：學到的觀念、書、資源。

這份筆記，會在實習中後期成為你的「超能力」。

## 三、實習中期（第 3–6 週）：價值期

### 1. 找一個「Quick Win」
在第 4 週前完成一件對團隊有貢獻的小事。例如：
- 自動化一個重複的 Excel 流程。
- 整理一份過去沒人整理的 SOP。
- 主動寫一份競品分析。
- 為團隊知識庫建立一個 Notion 主題頁。
- 找出某流程的 bottleneck 並提出解方。

這會讓主管發現「你和其他實習生不一樣」。

#### 真實範例
一位 PM 實習生發現團隊每週要花 3 小時手動整理「上週用戶回饋」。她寫了一個 Google Apps Script 自動化這個流程，每週節省 3 小時 × 4 人 = 12 工時。

主管在週會公開稱讚：「Sarah 解決了一個我們忍受了 6 個月的問題。」

她最後的 Return Offer 是 A+。

### 2. 主動承擔超出範圍的任務
當你聽到主管說「啊這個沒人做」、「這個拖很久了」——立刻說「我可以試試」。

#### 主動承擔的範例
- 「沒人寫的競品分析」→ 我可以做。
- 「沒人整理的會議紀錄」→ 我可以接。
- 「沒人 review 的 PRD」→ 我可以幫。

### 3. 每週寄「週報」給主管
即使主管沒要求，每週五寄一封：

#### 週報結構
```
Subject: [實習生週報] Week 3 - 進度與下週計畫

Hi 主管，

本週完成：
- [專案 A] 完成 X 部分，預計下週進入 Y 階段。
- [專案 B] 與設計師確認 wireframe，今晚進入開發。
- [其他] 整理出競品分析草稿（連結附上）。

本週學到：
- 學會 X 工具的進階用法。
- 從 Y 同事身上學到 Z 觀念。

下週計畫：
- 完成 [專案 A] 的 Y 階段。
- 開始 [專案 C]。

需要的支援：
- 是否方便週二一起 review [專案 A]？
- 對於 [某個決策] 我有 2 個想法，希望和您討論。

謝謝！
ZZ」
```

這種主動回報是 Return Offer 的隱藏神器。

### 4. 建立跨部門的「支持者」
不要只當「直屬主管的實習生」，要當「整個部門的好助手」。

#### 做法
- 主動幫設計師整理用戶資料。
- 主動為工程師準備 PRD 範本。
- 主動為業務蒐集競品資訊。

這樣 Return Offer 評選時，**會有 5 個人為你說話，不是只有 1 個**。

## 四、實習後期（第 7–10 週）：證明期

### 1. 完成一個「可被講述」的成果
最後一週前，你要有一件「能在面試中講 5 分鐘」的事。

#### 標準
- 你是主導者（不是助手）。
- 有量化成果。
- 對團隊 / 公司有實質影響。

### 2. 主動約主管「中期回饋」
「我這週剛好實習過半，想請您給我一些回饋，特別是哪些地方需要改進。」

**這展現的不是脆弱，是成長心態**。

#### 中期回饋對話
> 你：「我想請您給我一些回饋，特別是您覺得我哪些地方做得好、哪些需要改進。」
> 主管：「你做得很好，但 X 部分可以再加強。」
> 你：「謝謝您的回饋。針對 X，我打算 Y 來改進。下週我會再回報進度。」

這場對話結束後，你已經在主管心裡留下「會反思、會行動、會跟進」的印象。

### 3. 與 HR 主動溝通
不要等 HR 來找你，最後 2–3 週可以禮貌詢問：
> 「請問貴公司今年是否有 Return Offer 名額？需要我做什麼準備？」

### 4. 寫實習結束的「成果簡報」
製作一份完整的「實習成果簡報」（10–15 張投影片）：
- 我做了什麼。
- 帶來什麼影響。
- 學到什麼。
- 接下來的建議。

這份簡報會留在公司，成為決定 Return Offer 的關鍵之一。

## 五、影響 Return Offer 的關鍵指標

### 1. 工作表現
- 任務交付速度與品質。
- 主動性與創造力。
- 跨部門協作能力。

### 2. 文化契合度
- 和團隊互動是否自然。
- 是否認同公司價值觀。
- 是否願意長期投入。

### 3. 商業判斷力
- 能否從用戶 / 客戶角度思考。
- 能否平衡短期與長期。
- 能否在資源限制下做決策。

### 4. 學習力
- 能否快速吸收新工具。
- 能否從錯誤中調整。
- 能否主動找資源。

### 5. 影響力
- 是否帶來具體的改變。
- 是否影響到其他同事的工作方式。
- 是否被「記住」。

## 六、實習中容易踩的五大地雷

### 地雷 1：過度被動
等到主管來分配任務，從不主動。

**徵兆**：
- 每天等 task 來。
- 不問問題。
- 沒主動約 1on1。

**矯正**：每週主動找主管 1 次「我能不能做 X」的對話。

### 地雷 2：不問問題或問太多沒準備的問題
兩種極端都不好。

**正確做法**：
- 問問題前先 Google / 看內部文件。
- 把問題整理成清單，1on1 時集中問。
- 問「策略問題」（為什麼）多於「執行問題」（怎麼做）。

### 地雷 3：不建立人脈
只和直屬主管聊天，其他人完全不熟。

**矯正**：每週至少和 1–2 位非直屬同事吃午餐或喝咖啡。

### 地雷 4：太快談薪水
第一週就問 Return Offer 條件。

**矯正**：聚焦在做事，到實習剩 2–3 週時再禮貌詢問。

### 地雷 5：最後幾週鬆懈
以為要結束了就少做事，這正是評分期。

**矯正**：把最後 2 週當成「決勝期」，比第一週更認真。

## 七、就算沒拿到 Return Offer，也別氣餒

不少優秀的學生因為 HR 名額限制、組織重組等因素未獲 Return Offer，但他們的下一步：
- 用實習主管當推薦人。
- 把實習經驗精準寫進履歷。
- 兩個月內拿到其他公司 Offer。

> **沒有 Return Offer ≠ 表現不好**，但有 Return Offer ≈ 你已經是準正職員工。

### 沒拿到 Return Offer 的應對策略

#### 1. 主動詢問原因
> 「謝謝您過去幾個月的指導。如果方便，我希望了解具體哪些方面我可以再加強，以便未來改進。」

#### 2. 請求轉介紹
> 「您是否認識其他公司有類似職缺，方便為我介紹嗎？」

#### 3. 維繫關係
即使沒 Offer，主管也是未來的推薦人。每 3–6 個月更新近況。

## 八、Return Offer 後的進階談判

拿到 Return Offer 不是終點。你可以談：

### 1. 起薪
若你在業界已有比較基準（其他 Offer），可以爭取 5–15% 加薪。

### 2. 部門 / 團隊輪調
有些大公司允許你在簽約後選擇進入哪個團隊。

### 3. 報到日期彈性
若想畢業後出國旅遊或休息，可以爭取延後 1–2 個月。

### 4. Sign-on Bonus
頂尖公司常給 5–20 萬簽約金，特別是若你有競爭性 Offer。

### 5. 升遷時程承諾
能否討論第一次績效檢視時間點。

## 九、實習中期的「自我檢核」

### 第 4 週自問
- 我和主管的 1on1 都被善用嗎？
- 我認識的同事超過 5 位嗎？
- 我有 1 個「即將完成」的具體成果嗎？

### 第 6 週自問
- 我是否有主動承擔的紀錄？
- 我是否有跨部門的合作？
- 我是否被同事「記住」（不是只是「實習生」這個角色）？

### 第 8 週自問
- 我能用 5 分鐘講清楚我這次實習的成果嗎？
- 我有 3 個能寫進履歷的量化結果嗎？
- 我有 2 位以上願意幫我寫推薦信的人嗎？

## 十、Return Offer 的「跨產業差異」

### 顧問業
- 評估標準：思考力、簡報力、客戶溝通。
- Return Offer 多半 8 週後立刻決定。

### 投資銀行
- 評估標準：細節控、抗壓性、團隊配合。
- 「拿到 Return Offer = 入行門票」。

### 科技業
- 評估標準：產品思維、跨部門協作、學習力。
- 部分公司 Return Offer 可延後決定（畢業前才確認）。

### 廣告 / 媒體
- 評估標準：創意執行、客戶服務。
- Return Offer 比例較低，需要主動爭取。

## 十一、FAQ

**Q1：實習表現很好但公司沒 Return Offer 名額怎麼辦？**
請主管寫推薦信，並主動詢問是否能介紹其他公司。

**Q2：可以同時申請多家 Return Offer 嗎？**
通常不行。多數公司有 exclusivity 條款。

**Q3：Return Offer 一定要接受嗎？**
不一定。你可以拒絕（特別是發現公司不符你期待）。

**Q4：拿到 Return Offer 還能找其他工作嗎？**
看合約。多數允許，但時間有限制。

**Q5：實習表現一般可以拿 Return Offer 嗎？**
看公司。部分公司有「保底名額」，但好的職位、團隊、薪資只給頂尖實習生。

## 十二、提升 Return Offer 機率的「終極清單」

實習開始前：
- [ ] 認識公司的 3 個目標客戶。
- [ ] 讀公司過去 6 個月的新聞。
- [ ] 列出 5 個我想學到的能力。

第 1 週：
- [ ] 與主管做期待對齊。
- [ ] 認識所有團隊成員。
- [ ] 建立筆記系統。

第 2–4 週：
- [ ] 完成第一個 Quick Win。
- [ ] 開始每週寄週報。
- [ ] 至少跨部門合作 1 次。

第 5–6 週（中期）：
- [ ] 主動約主管 mid-term feedback。
- [ ] 確認自己的「故事」開始成形。
- [ ] 至少和 5 位同事建立關係。

第 7–8 週（後期）：
- [ ] 完成可講述的核心成果。
- [ ] 主動詢問 Return Offer 流程。
- [ ] 開始準備成果簡報。

第 9–10 週：
- [ ] 進行成果簡報。
- [ ] 確認 Return Offer 結果。
- [ ] 寫感謝信給所有合作過的同事。

## 結語

Return Offer 不是運氣，而是「做對的事 × 持續做」的結果。當你把實習當成「為期 10 週的真實工作試用」，你的態度會自然變得不同。

**核心原則**：
1. **第一週定錨**：問清楚「成功是什麼樣子」。
2. **持續主動**：每週都比上週做得多一點。
3. **建立支持者網絡**：不只討好主管，要討好整個團隊。
4. **最後幾週是決勝期**：不要鬆懈，要全力衝刺。

**行動建議**：到實習通查找你目標公司的「轉正比例」與「轉正關鍵」相關真實心得，提早準備你的轉正策略。把這份檢核清單列印出來，貼在你的書桌前，每週對照進度。
