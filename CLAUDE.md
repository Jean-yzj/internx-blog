# InternX Blog

Next.js 14 blog covering internship guidance and career development for Taiwanese students.

## Stack
- Next.js 14 (Pages Router)
- React 18
- Tailwind CSS + @tailwindcss/typography
- Markdown content via `gray-matter` + `remark`
- Node 18.17+
